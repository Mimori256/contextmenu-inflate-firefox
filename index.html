Firefox Repro: Native Context Menu Inflates Event Timing Duration

Goal: Demonstrate that in Firefox, opening the native context menu can inflate PerformanceEventTiming.duration (and therefore INP candidate values) by roughly the amount of time the menu stays open.
How to reproduce:
  1. Open this file in Firefox.
  2. Open DevTools Console (optional; logs are also shown on this page).
  3. Right-click inside the striped box below.
  4. Keep the native context menu open for 1-2 seconds. Do not move the mouse cursor while the menu is open.
  5. Close the context menu and observe logged entries.
  6. In the observed entries, the presentationDelay should be roughly the amount of time the menu was open (unless you moved the cursor while the menu was open)
